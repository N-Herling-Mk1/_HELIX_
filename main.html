WHAT IS THIS?
STATIC VIEWING ELEMENTS
BE / FE LED indicators show the live binary state of the Flask backend server on localhost:5000 and the static frontend on GitHub Pages. The poll timers display how long ago each was last checked — reset to 00:00 on every successful ping. The system clock gives a UTC-anchored local timestamp, updated every second. All three are passive readouts — they update automatically and require no interaction.
UI CHECKOUT
The BACK END indicator shows whether the Flask server is live on localhost:5000 — view only. The GoatCounter analytics button opens the external dashboard for live visitor stats, page views, referrers, and browser data — no cookies, GDPR compliant. This panel is otherwise read-only — no settings are changed here.
