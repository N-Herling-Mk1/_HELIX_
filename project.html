PHYSICS CONTEXT

The Standard Model does not account for dark matter. The Hidden Sector Scalar (HSS) extension introduces light neutral scalars S that mix with the Higgs and decay with macroscopic decay lengths — called Long-Lived Particles (LLPs).

At ATLAS, LLPs produced in pp collisions at √s = 13.6 TeV can travel meters before decaying, leaving displaced tracks inside the Muon Spectrometer. The MSVtx signature — a cluster of MS tracks converging at a displaced point — is the experimental handle. This search targets H → SS → bbbb with S masses in the 5–55 GeV range.

ANALYSIS PIPELINE
01
NTUPLE INGESTION
Run 3 MC ntuples on atlng01.physics.arizona.edu. AnalysisBase 25.2.81. Signal MC (4 mass points) + data24VR + PunchThrough backgrounds.
02
CUTFLOW — CutflowRunner.cxx
Sequential MSVtx selection via RDataFrame. Key cuts: MSVtx multiplicity, HTmiss > 40 GeV (MeV units in ntuples), segment counts, quality flags. df11 = final stage.
03
SNAPSHOT → CSV
ROOT Snapshot after df11: 4 flat files per mass point (nn1/nn2 × barrel/endcap). convert_snapshots_to_csv.py via PyROOT → pandas CSVs. mcChannelNumber skipped for data.
04
MK3 FOCUSED MLP TRAINING
Four MLPs (128→128→1, sigmoid). k-fold CV for data sensitivity. Bayesian HMC for parameter uncertainty. RRM penalty: [1−AUC, fold_variance, σ_HMC].
05
INFERENCE → P(signal)
Trained mk3 models produce P(signal) score distributions on held-out test sets and background CSVs. ABCD plane closure check as 4th RRM component pending.
06
MK4 SSL TARGET — HELIX Framework
Self-supervised contrastive learning over cutflow sequences. Tokenization strategies: S1 object-level, S2 track-level, S3 cutflow-stage. InfoNCE with τ = 1/ΔS.
